# Task 6 - Sales Trend Analysis Using Aggregations

## Project Title

Walmart Sales Trend Analysis Using SQL

## Objective

Analyze monthly revenue and order volume using SQL aggregate functions and grouping techniques.

## Dataset Used

Walmart Sales Dataset

## Tools Used

* SQL
* MySQL / SQLite
* Walmart Sales Dataset

## SQL Concepts Covered

* SELECT
* GROUP BY
* ORDER BY
* SUM()
* COUNT(DISTINCT)
* Aggregate Functions

## Analysis Performed

### Monthly Revenue Analysis

Calculated total revenue generated each month using SUM().

### Monthly Order Volume Analysis

Calculated total orders per month using COUNT(DISTINCT invoice_id).

### Top Sales Months

Identified the highest-performing months based on revenue.

## Key Insights

* Monthly sales trends were analyzed.
* Revenue varied across months.
* Order volume patterns were identified.
* Top-performing months generated the highest revenue.

## Outcome

Successfully used SQL aggregations to analyze sales trends and business performance over time.
